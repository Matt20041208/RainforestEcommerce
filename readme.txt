雨林电商项目
技术栈：Spring Boot、MyBatis、MySQL、Redis、Lombok、PageHelper、Spring Security、JWT、Spring Security Filter
项目描述：构建了一套企业级电商解决方案，采用前后端分离架构，实现了用户购物与商家管理的完整业务闭环。系统支持用户注册登录、商品浏览与搜索、购物车管理、订单创建与追踪、地址管理等核心功能；商家端提供商品发布、库存管理、订单处理、数据统计等运营工具。通过JWT令牌机制实现无状态身份认证，利用Redis缓存热点数据优化系统性能，采用MyBatis实现高效的数据持久化，整体架构清晰、扩展性强，具备良好的可维护性。
